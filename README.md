# XYDSignTool

XYDSignTool 是一套面向 AutoCAD 的效率插件，主要用于标识系统算量、图纸目录管理、批量出图、图框识别、统计测量和批量嵌图等制图流程。

本项目由 **玄宇绘世设计工作室** 免费开源发布。

GitHub 仓库地址：<https://github.com/XYD-Studio/XYDSignTool>

## 功能概览

- 图库中心：插入标识属性块、标准图框属性块。
- 自定义图框识别：适配用户公司已有属性块图框，可配置块名前缀和图名、图号、比例、图幅字段映射。
- 标识系统算量：识别图库中心标识图块，生成数量表、Excel、JSON 和概算清单。
- 出图与目录：识别模型空间和布局空间图框，批量导出 PDF，管理图纸目录数据。
- 统计与测量：图块计数、动态块长度统计、规格统计、线段总长、批量查字段、批量旋转。
- 图片工具：批量嵌入图片到 CAD 图纸中。
- 帮助页面：内置插件说明、使用教程、定制开发联系方式和版权声明。

## 适用场景

- 铁路/高铁车站标识施工图设计。
- 地铁车站标识施工图设计。
- 标识导视图纸整理与算量。
- 批量图框目录生成。
- 批量 PDF 出图。
- 公司已有属性块图框的识别和适配。

## 基本使用

1. 在 AutoCAD 中加载 `XYDSignTool.dll`。
2. 切换到 `XYD 工具集` Ribbon 选项卡。
3. 按需求使用图库中心、标识系统算量、出图与目录、统计与测量、图片工具等功能。
4. 如使用公司自有图框，先通过 `图框识别` 配置识别规则。
5. 批量出图前，建议先检查图框中的图名、图号、比例、图幅属性是否完整。

## 标识算量说明

标识系统算量通过识别图库中心插入的标识图块进行统计。插入标识图块后，可以修改属性，系统会自动识别。

属性块中如果没有数量属性，则默认数量为 `1`。

如果需要用一个编号表达多个连续标识，请使用英文波浪线 `~` 连接，例如：

```text
ZF-XQ-01~03
```

该编号会被识别为 3 个标识。请注意，其他连接符号无效。

系统中的单价仅供参考，不作为结算、审计或其他正式依据。

## 图框与目录说明

插件自带标准图框属性块，也支持适配用户公司已有图框。

插入图框后，可以将图框中的图号、图名、比例等属性文字手动调整到公司图框中对应的位置，但不要炸开图框，否则属性块信息可能丢失，影响后续识别、目录生成和批量出图。

如果图幅不在属性中，也可以从图块名称中提取，例如 `A0`、`A1`、`A2+2` 等。

## 编译环境

当前项目为 AutoCAD .NET 插件工程，目标框架：

- .NET Framework 4.8
- x64
- AutoCAD 2024 相关 .NET 引用

Debug 编译命令示例：

```powershell
dotnet msbuild XYDSignTool.sln /t:Build /p:Configuration=Debug /p:Platform=x64 /v:minimal
```

编译输出：

```text
bin\x64\Debug\XYDSignTool.dll
```
